## 22:34 真正根因：Vite 代理配置错误

### 用户反馈
"仪表盘的所有图表都没有显示"

### 真正根因 (3次才找到)
**Vite proxy 配置错误**：vite.config.js 的 proxy target 指向了 `localhost:8080` (typo)
- 所有 `/api/*` 请求都返回 502 Bad Gateway
- 浏览器侧 axios 收到 502 → catch 静默处理 → 图表数据空
- 之前 console.log 都没输出（请求都没到达后端）

### 修复
1. **vite.config.js (CRM)**: proxy target `8080` → `8081`
2. **vite.config.js (Portal)**: 添加 proxy 配置（之前没有）
3. **request.js (CRM)**: baseURL 改 `/api` (走proxy)
4. **request.js (Portal)**: baseURL 改 `/api/member-portal`
5. **ProductList.vue**: uploadUrl 改 `/api/upload`

### 验证
6个图表端点全部通过Vite proxy返回数据。推送 commit 9d08325。
